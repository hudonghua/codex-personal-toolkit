1A. 算法PCB接收车体姿态点

车体姿态确定界面固定下发 O/Q1/Q2/Q3/Q4 五帧。算法PCB收到五点后自动检查点间距离和 R/T 残差；有 Last Good 时设防，无 Last Good 时建立初始有效点库。算法PCB只接收局部相对坐标；N_O/E_O/Z_O 是上位机本次 O 点的全站仪绝对坐标，由上位机保存和显示。

通信方向CAN ID分包Byte定义算法PCB处理
电脑 -> 算法PCB0x50B7=0xA1B0点序号0=O，1=Q1，2=Q2，3=Q3，4=Q4；5 点齐后自动校验。
电脑 -> 算法PCB0x50B7=0xA1B1-B2dN int16低字节在前；dN=N_point-N_O，全站仪绝对 N/E/Z 由上位机先减当前 O，只下发局部坐标。
电脑 -> 算法PCB0x50B7=0xA1B3-B4dE int16低字节在前；dE=E_point-E_O。
电脑 -> 算法PCB0x50B7=0xA1B5-B6dZ int16低字节在前；dZ=Z_point-Z_O。

1B. 算法PCB接收机械误差修正点

机械误差修正按单臂执行。上位机可以长期维护采样池；算法PCB只接收当前启用点集。左臂标定时，不能改动中臂/右臂参数，也不能更新全局 R/T。H 点坐标也使用上位机保存的当前 O 点绝对坐标作基点，算法PCB不保存全站仪绝对大坐标。

通信方向CAN ID分包Byte定义算法PCB处理
电脑 -> 算法PCB0x50B7=0xA2B0对象+点序号0x00-0x3E 左臂点；0x40-0x7E 中臂点；0x80-0xBE 右臂点。只写对应臂临时缓存。
电脑 -> 算法PCB0x50B7=0xA2B0完成标记0x3F 左臂完成；0x7F 中臂完成；0xBF 右臂完成。收到后校验/拟合该臂当前点集。
电脑 -> 算法PCB0x50B7=0xA2B1-B2dN int16低字节在前；H 点全站仪坐标也先减当前 O，dN=N_H-N_O。
电脑 -> 算法PCB0x50B7=0xA2B3-B4dE int16低字节在前；dE=E_H-E_O。
电脑 -> 算法PCB0x50B7=0xA2B5-B6dZ int16低字节在前；dZ=Z_H-Z_O。
