## 3. 正解核心：`arm_calc_one()`

采用 **DH 风格的齐次变换**：固定连杆向量 + 关节旋转/平移，逐级累乘旋转矩阵。

### 输入（每条臂 `ARM200A_INPUT`）

| 字段 | 含义 |
|------|------|
| `root_deg` | 根部绕 X（仅左/右臂） |
| `base_deg` | 基座回转（绕 Z） |
| `boom_deg` | 大臂摆动（绕 Y） |
| `cd_mm` | C→D 伸缩 |
| `d1_deg` | D1 级转 |
| `e1_deg` | E1 级转（绕 `ee1` 轴） |
| `f_deg` | F 级转（绕 `e1e2-e1f` 轴） |
| `gg1_mm` | G→G1 伸缩 |

### 几何基线 + 标定修正

- **基线**：`gArmLeftBase / gArmMidBase / gArmRightBase`（Creo 名义尺寸）
- **修正**：`gArm*Fix`（11 个 3D 向量 + `dd1_mm`，LM 拟合得到）

实际连杆向量 = `base向量 + fix向量`。

### 递推公式（概念）

```text
O0 = 固定锚点
O1 = O0 + Rx(root)·v_o0o1        （有 O0 时）
O  = O1 + v_o1o
B  = O  + v_ob
A  = B  + Rz(base)·v_ba
C  = A  + Ry(boom)·v_ac
D  = C  + Ry(boom)·(cd_mm, 0, 0)
D1 = D  + Ry(boom)·(dd1_mm, 0, 0)
E  = D1 + Ra(局部X, d1)·v_d1e
E1 = E  + v_ee1
F  = E1 + Ra(ee1轴, e1)·v_e1f
G  = F  + Ra(f轴, f)·v_fg
G1 = G  + Ra(f轴, f)·(gg1_mm, 0, 0)
H  = G1 + Ra(f轴, f)·v_g1h
H1 = H  - Ra(f轴, f)·(300, 0, 0)   ← 刚性附加点，非独立关节
```

旋转实现：`arm_rx/ry/rz`（基本轴）+ `arm_ra`（任意轴 Rodrigues 公式）。

## 4. Body → World 坐标转换

正解只在 **车体 Body 系** 算几何；摆车/现场摆放单独做：

```text
p_world = R_body→world · ( Ry(pitch)·Rx(roll) · p_body ) + t
```

对应 `arm_apply_coord_point()`：

1. 用 `body_roll_deg`、`body_pitch_deg` 做双轴倾斜补偿  
2. 再乘 Q1~Q4 标定得到的 `gArmCoordTrans`（Body→World 刚体变换）

`h` 与 `h1` 必须一起走同一套变换，否则大地系下 H1→H 方向会错。
