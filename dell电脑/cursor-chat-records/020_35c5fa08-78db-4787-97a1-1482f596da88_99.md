```text
在 C 里，**函数调用**就是：在表达式或语句里写出**函数名**，后面跟一对圆括号，括号里按顺序传入**实参**（参数个数、类型要和声明/定义一致）。

## 基本形式

```c
返回值类型 函数名(形参类型 形参名, ...);   /* 声明（原型），让编译器知道怎么检查调用 */

/* 调用： */
函数名(实参1, 实参2);
x = 函数名(实参1);   /* 有返回值时，可以参与赋值或表达式 */
```

- **无参函数**：`foo();`（在 C 里不要写成 `foo` 不带括号，那不是“调用”。）
- **有返回值**：调用表达式的类型就是函数的返回类型；若返回 `void`，不能拿去赋值。

## 实参怎么传进函数（要点）

C 里**默认是按值传递**：

- 传**普通变量**：函数里得到的是**一份拷贝**，改形参**不会**自动改外面的变量。
- 若要在函数里**修改调用者的变量**，要传**指针**，例如 `void swap(int *a, int *b)`，调用时写 `swap(&x, &y)`。
- **数组名**在多数情况下会**退化为指向首元素的指针**，所以像 `void f(int a[])` / `void f(int *a)` 这类，本质上常按指针方式传。

## 调用前需要“知道长什么样”

- 若**定义写在调用之后**，前面要有**函数原型**（声明），否则旧标准/某些情况下容易出问题；现代写法通常是：头文件里声明，一个 `.c` 里定义。
- **递归**就是函数体里再次调用自己，要有终止条件，否则会栈溢出。

## 和“链接”的关系（简略）

- **声明**告诉编译器如何检查调用、如何生成调用代码。
- **定义**提供真正的函数体；最终由**链接器**把各 `.o` 里的符号拼起来。只声明没定义会在链接阶段报错（`undefined reference` 一类）。
