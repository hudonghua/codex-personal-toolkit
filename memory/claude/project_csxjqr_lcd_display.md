## 数据源

### 压力数据
- **来源：** `gRunInfo.vRcvWL[0-6]`
- **模式切换：** `gRunInfo.vRcvWL[6]`
  - ==1: 显示小臂/变幅压力
  - ==2: 显示油泵/回转压力
- **数值组合：** `vPress = vRcvWL[n] + vRcvWL[n+1]*256`
- **显示转换：** `vPress/10` + "." + `vPress%10`

### 状态报警
- **来源：** `gRunInfo.vRcvWL[7]` 的各个bit位
- **显示逻辑：** bit为1显示文字，为0显示空格清除

---

## 刷新控制（BF0-BF7变量）

### 变量对应关系
- `BF0` ↔ `gRunInfo.vRcvWL[0]`
- `BF1` ↔ `gRunInfo.vRcvWL[1]`
- ...
- `BF7` ↔ `gRunInfo.vRcvWL[7]`

### 刷新逻辑
1. **压力标签：** 只在 `gLcdFresh==2`（进入界面）时显示一次
2. **压力数值：** 当 `vRcvWL[n] != BFn` 时刷新
3. **状态报警：** 当 `vRcvWL[7]` 对应bit变化时刷新
4. **模式切换：** 当 `vRcvWL[6] != BF6` 时，强制刷新所有压力数值（BF0-BF5设为0xFF）
