## 接口概述

**屏型号**：7寸屏200A接口定义  
**接口类型**：
- 35芯插头（AMP35）- 主接口
- 8芯插头 - 辅助接口

**资源总览**：
- 12DO + 12DI（其中2路低有效）
- 1路4-20mA模拟量
- 2路0-1k欧模拟量 + 3路可复用DI
- 总计6路模拟量采集

## 功能分组总结

### 推进系统
- DO1 (PWM3-A)：推进前进
- DO6 (PWM3-B)：推进后退
- DI1-H / PI：推进反馈

### 旋转系统
- DO2 (PWM4-A)：旋转正转
- DO3 (PWM4-B)：旋转反转
- DI4-H/L：旋转压力采集

### 冲击系统
- DO4 (PWM5-A)：冲击方向DT101
- DO5 (PWM5-B)：Y121液压马达
- DI9-H：液压启动

### 液压系统
- DO12 (PWM6-A)：液压泵启动
- DO11 (PWM6-B)：水泵
- 模拟量采集：液压压力、水压、气压等

### 辅助系统
- DO7 (PWM1-B)：喷水控制
- DO8 (PWM1-A)：稳压阀Y122
- DO9 (PWM2-B)：冷却风扇
- DO10 (PWM2-A)：冲击泵启动

### 模式和启动
- DI5-H：模式选择
- DI10-H：钻机
- DI13-H：钻机启动
- DI14-H：水泵
- DI15-H：液压启动停止

## PWM分组规则

### PWM组分配
| PWM组 | A通道 | B通道 | 功能 |
|------|------|------|------|
| PWM1 | DO8 | DO7 | 稳压阀/喷水 |
| PWM2 | DO10 | DO9 | 冲击泵/冷却风扇 |
| PWM3 | DO1 | DO6 | 推进前/后 |
| PWM4 | DO2 | DO3 | 旋转正/反 |
| PWM5 | DO4 | DO5 | 冲击方向/液压马达 |
| PWM6 | DO12 | DO11 | 液压泵/水泵 |

**约束**：同组A/B可分别绑定正反动作，但同一时刻不能同时输出

## 模拟量采集规范

### 采集通道
```
AD0：4-20mA信号（液压泵、冲击压力）
AD1：0-5V信号（水压）
AD2：0-5V信号（旋转压力、液压）
AD3：0-5V信号（稳压阀压力）
AD4：0-5V信号（液压压力）
AD5：0-5V信号（液压压力0.3Mpa-2.5Mpa）
```

### 数据转换
- 4-20mA范围：0-4372mV
- 0-5V范围：0-5000mV
- 欧姆范围：0-1000欧（对应0-888）

## 接口使用建议

### 电源
- 工作电压：10-32V
- 多个GND脚确保接地可靠

### CAN总线
- 两套独立CAN（CAN1和CAN2）
- 8芯插头也有CAN接口，可用于扩展

### 模拟量
- 6路模拟量采集，支持混合类型（4-20mA、0-5V、欧姆）
- 部分通道可复用为DI

### PWM输出
- 12路DO可复用为6组PWM
- 同组内A/B互斥，不能同时输出
