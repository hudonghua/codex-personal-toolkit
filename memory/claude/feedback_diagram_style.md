**规则：** 所有图表和HTML原理图使用清爽蓝色风格展示

**Why:** 用户明确指定这个风格作为标准，清爽蓝色风格具有更好的对比度、更大的字体、更清晰的视觉层次

**How to apply:** 
- 创建任何HTML原理图、框图、接线图时，使用以下样式作为基础：
  - pin-row 背景：#e3f2fd（浅蓝），边框：#64b5f6（蓝）
  - pin-number：#1976d2（深蓝）背景，白色文字，14px 字体，8px padding
  - pin-name：#1565c0（蓝），16px 字体，bold
  - pin-function：#d32f2f（红），#fff3e0（浅橙）背景，15px 字体，font-weight 600，8px padding
  - 输入侧头部：#0066cc（蓝）
  - 输出侧头部：#00aa00（绿）
  - 悬停效果：背景 #bbdefb，边框 #2196f3
