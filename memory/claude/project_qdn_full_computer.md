## 系统架构

**三层架构**:
1. QT视觉校准层 - 相机识别四点光源，solvePnP解算位姿
2. 控制器坐标变换层 - 两个固定棱镜+倾角传感器，设备坐标→大地坐标
3. 控制器正运动学层 - 7自由度运动链计算

**7个自由度**:
- B: 回转角 (-20° ~ +20°)
- A: 俯仰角 (0° ~ 50°)
- C→D: 大臂伸缩 (0 ~ 2000mm)
- D1: 关节角 (-180° ~ +180°)
- E: 自转角 (-80° ~ +80°)
- F: 推进梁俯仰 (-30° ~ +30°)
- G→G1: 推进梁伸缩 (0 ~ 1000mm)
